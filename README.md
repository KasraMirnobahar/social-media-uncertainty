# Social Media Uncertainty Demo

This repository contains a local Python demo task developed as part of a behavioural research project on social media uncertainty and checking behaviour.

The task simulates a social media-like environment in which users interact with meme-based content. The broader purpose of the project is to explore how uncertainty in social media reward timing may influence checking behaviour.

This repository is currently intended as a **demo version** of the task. It does not collect or save participant data.

## Project Aim

Social media platforms often deliver rewards, such as likes, notifications, or feedback, with varying degrees of predictability. This demo is designed to represent a simplified experimental environment for studying how reward timing uncertainty may affect checking behaviour.

The project may later be developed into a fuller behavioural experiment or reusable research tool.
